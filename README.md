# 介绍
本目录下代码用于介绍使用TopsInference2.0接口搭建完整的推理应用


## 目录结构如下

```shell
├── cpp          --cpp sample code
├── data         --data for testing
├── engines      --engine file
├── models       --preset model for evaluation & integration
├── python       --python sample code
└── README.md    
```
